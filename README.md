# Macondo---Calculator
This is the first project I made for Macondo, and it's very important to me because I made it. 
Basically, I used HTML, CSS, and JS to build, style, and make the app function. The color contrast (black and a decent blue) makes it look lavish and addicting.
I used Gemini to help correct my variables and missing logic, and watched a YouTube JavaScript tutorial to code the functionality of the calculator.
# Languages
HTML: Setting up the input box, buttons for the operators, numbers and etc. With that, the linking of JS and CSS and Title for the APP.
CSS: The design and style of the calculator, the beautiful color contrast made with it. It was my fav part of coding the app.
JS: To actually make the calculator work and as a beginner, it was the most diffcult part to code due to its variable names.
# Reason to make this
I always loved watching people code and started learning HTML at first. Out of curiosity, I also learned parts of CSS and a little bit of java to make a real working app as alone the face isnt the thing. Now that I knew very much and had watched tutorials it was easier as a beginner coder to code the app . I chose calculator as I love maths and also cuz it was easier for me to code with my little experience.
# Features
Additon, Subtraction, Equating, Index power solving, Multiplication, Division, Factorials, Cleaning the queue ( AC ), cutting of a character ( DEL )
